This folder is created to hold the reports:
├── aws_missing_files.txt: the list of PCAWG files missing from AWS
├── aws_missing_index_files.txt: the list of PCAWG files missing index files from AWS 
├── collab_missing_files.txt: the list of PCAWG files missing from Collab
├── collab_missing_index_files.txt: the list of PCAWG files missing index files from Collab 
├── dcc_portal_missing_files.txt: the list of files missing from DCC portal
├── fileobject.jsonl: file object JSONL dump
├── files_only_exist_in_one_repo.txt: the list of files_only_exist_in_one_repo
├── index_files_missing_in_collab_existing_in_aws.txt: the list of index files missing from Collab existing in AWS
└── overall_report.txt: statistic summary for each repository
